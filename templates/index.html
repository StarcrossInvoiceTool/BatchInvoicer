CSV Format Guide

Your CSV should include the following columns:

  • Patient Info: patient_name, patient_address, patient_postcode
  • Invoice Info: invoice_number, invoice_date, account_ref, ref, po_number, payment_terms, period
  • Financial Info: net, discount, subtotal, vat_amount, total
  • Bank Details: bank_name, bank_account_name, bank_account_number, bank_sort_code
  • Invoice Items: date, our_ref, client_ref, nhs_number, contract_hospital, booked_by, from_location, to_location, status, directions, mob, wait_pounds, wait_notes, miles, charged, miles_pounds, job_pounds, total

Note: The first row should contain metadata (patient, invoice, financial, bank info). Subsequent rows should contain invoice items. Each row after the first represents one invoice item.
